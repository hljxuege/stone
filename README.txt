1.管理后台
2.andriod客户端

业务设计
1.业务申请，流程通过网页 点击“我要申请” ，后台 人工受理过程[查看][接受][拒绝]
受理条件，1注册，2申请描述，3地理信息，4联系电话
2.订单受理， 订单状态[收][进行][送] [价格][实际收款]

辅助后台设计
分配商户账号， 每个商户账号彼此独立



细节
用户 三种类型  1商户[限制员工创建数量]， 1.1商户外派人员[商户员工]，2客户， 3系统管理员
消息系统：
   业务申请
   订单消息 [订单号]
